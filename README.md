## Proyecto Netflix

Analisis de datos de peliculas y series para entender el comportamiento de los usuarios a lo largo del tiempo. Datos desde TMDB.

## Estructura

- src: codigo
- data: ficheros crudos y trabajados
- notebooks: ficheros de prueba

## Instrucciones

- Virtual env: python -m venv.venv
- Activarlo: .venv\scripts\activate
- Librerias: pip install -r requirements.txt
